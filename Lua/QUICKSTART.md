# 🚀 Início Rápido - Lua NPC

## ⚡ Como Começar em 1 Minuto

### Windows

```bash
# Dê duplo-clique em:
play.bat
```

### Linha de Comando

```bash
lua main.lua
```

### Com Lua 5.3

```bash
lua5.3 main.lua
```

---

## 🎮 Escolha seu NPC

Ao iniciar, você pode escolher entre:

1. **Robin** 😊 - Amigável
2. **Shadow** 🌑 - Misterioso
3. **CORE** 🤖 - Profissional
4. **Blaze** 🔥 - Entusiasta

---

## 💬 Sistema de Diálogos

Cada NPC possui:

- ✅ Saudações únicas
- ✅ Reações personalizadas
- ✅ Dicas estratégicas
- ✅ Comemoração na vitória
- ✅ Consolo na derrota

## 💡 Usando Dicas

Você tem **3 dicas** por partida!

Durante o jogo, digite:

```text
📝 Digite uma letra (ou 'dica' para dica): dica
```

---

## 📋 Controles

| Ação | Comando |
| ------ | --------- |
| Adivinhar letra | `[A-Z]` |
| Obter dica | `dica` |
| Jogar novamente | `s` |

## 🐛 Problemas?

### "Lua command not found"

Instale Lua de [https://www.lua.org/download.html](https://www.lua.org/download.html)

### "Caracteres estranhos"

Use PowerShell no Windows ou terminal Linux

### "Script não funciona"

Verifique:

```bash
lua -v
lua5.1 -v
lua5.3 -v
```
