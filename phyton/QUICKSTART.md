# 🚀 Início Rápido - Python

## ⚡ Como Jogar Agora

### 1️⃣ **Se Python já estiver instalado:**

**Opção A - Windows (Mais Fácil):**
```bash
# Dê duplo-clique em:
play.bat
```

**Opção B - Linha de Comando:**
```bash
python script.py
```

### 2️⃣ **Se Python NÃO estiver instalado:**

Veja o guia [INSTALL.md](INSTALL.md) para instruções completas de instalação.

---

## 🧪 Teste Antes de Jogar

Para verificar se tudo está funcionando:

```bash
python test.py
```

Você deve ver algo como:
```
✓ Versão Python: 3.11.0
✓ asyncio disponível
✓ random disponível
✓ winsound disponível
✓ script.py encontrado
✓ TUDO OK! Pronto para jogar.
```

## 🎮 Controles do Jogo

- **Digite uma letra** e pressione ENTER
- **Acerte 6 letras** para vencer
- **Pressione `Ctrl+C`** para sair a qualquer momento

---

## 🔊 Áudio

- ✅ **Som em Windows:** Automático com `winsound`
- ⚠️ **Som em Linux/Mac:** Desativado (use versão web)
- 🎵 **Melodia:** Toca continuamente no fundo
- 🔉 **Efeitos:** Acerto, erro, vitória e derrota

---

## ❓ Perguntas Comuns

**P: O jogo não inicia?**
- R: Instale Python e reinicie seu computador

**P: Não ouço som?**
- R: Você precisa de Windows. Ou use a versão web em `html/index.html`

**P: Como saio do jogo?**
- R: Pressione `Ctrl+C` ou deixe perder todas as tentativas

**P: Posso adicionar mais palavras?**
- R: Sim! Edite a lista `PALAVRAS` em `script.py`
